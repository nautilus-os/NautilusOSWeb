About NautilusOS

NautilusOS is an innovative web-based operating system that brings the full desktop experience to your browser. Boot it up, log in, and interact with it just like a real desktop environment.

Everything runs client-side with no backend or external data storage. Built with pure HTML, CSS, and JavaScript—no frameworks, no dependencies except for icons and fonts.

Key Features

Desktop Environment

Fully animated desktop experience with draggable and resizable windows, taskbar, searchable start menu with app grid, custom right-click context menu, and complete window management including minimize, maximize, and close functionality.

Advanced Cloaking

Sophisticated cloaking options not seen in other proxies. Features include tab blackout, anti-monitoring tools, customizable blackout delays, and the ability to set custom tab titles and icons for extra discretion.

Customization

Extensive theming system with tons of prebuilt themes. Exportable and importable profiles for backup and sharing. Adjust window transparency, enable/disable animations, and personalize your experience.

AI Assistant

Built-in AI assistant integrated directly into the OS. Capable of answering questions, helping with tasks, and even controlling system functions with your explicit approval for automation.

Multi-User

Create and manage multiple user accounts with different editable permissions. Choose between superuser with full access or standard users with restricted capabilities. Each user gets their own desktop, settings, and app preferences.

App Ecosystem

Includes preinstalled apps and a built-in app store to discover more. Features include media player, screenshot tools, file explorer, notifications, quick action menu, and more. Install apps and extend functionality easily.

Notifications

Built-in notification system keeps you informed about system events, app updates, and important messages. Quick action menu provides instant access to common tasks and settings.

File Explorer

Full-featured file explorer for managing your virtual files and folders. Create, delete, move, and organize files with an intuitive interface similar to traditional desktop OS file managers.

Media Tools

Built-in media player for audio and video files. Screenshot capture tool for saving moments. Image viewer for photos. Full media suite without leaving Nautilus.

Built-in Games

Collection of preinstalled games for entertainment when you need a break. Play directly within the OS without external dependencies or downloads.

Profile Management

Export and import your complete Nautilus profile including settings, themes, apps, and preferences. Backup your setup or share configurations across devices easily.

Three Proxy Browsers

NautilusOS includes three different proxy browsers, each using different technologies to ensure you can always access the content you need.

Nautilus Browser

The primary browser built into the OS with optimized performance and integrated features.

Ultraviolet

Advanced proxy technology for bypassing restrictions with enhanced security.

Helios

Alternative browser using different proxy technology for redundancy and accessibility.

Getting Started

OneFile Version

Perfect for when Nautilus is blocked by your school or organization. Download the index.html file from the NautilusOS-OneFile folder and open it directly in your browser.

Note: Nautilus Browser is disabled by default in the file, but can be re-enabled in experimental settings. Ultraviolet browser won't work due to service worker limitations in file:// URLs.

Deploy Your Own

Create your own instance of NautilusOS with free hosting platforms:

  • Vercel - One-click deployment
  • Netlify - Simple setup
  • GitHub Codespaces - Temporary instance

GitHub Codespaces Instructions:

1. Click green code button → codespaces tab → "Create codespace on main"

2. Run python3 -m http.server 8080 in terminal

3. Make port 8080 public → click the forwarded address

4. Extend lifetime to 240 mins in profile settings
